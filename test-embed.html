🏢 NK Camp Interactive Chart - Iframe 임베딩 테스트

1. 기본 라이트 테마

기본 설정으로 밝은 테마를 사용한 차트입니다.

<iframe src="./embed.html" width="100%" height="600px" frameborder="0"></iframe>

2. 다크 테마

어두운 테마로 설정된 차트입니다.

<iframe src="./embed.html?theme=dark" width="100%" height="600px" frameborder="0"></iframe>

3. 컨트롤 없는 버전

캠프 선택 컨트롤을 숨긴 버전입니다.

<iframe src="./embed.html?controls=false&height=500px" width="100%" height="500px" frameborder="0"></iframe>

4. 반응형 임베딩

CSS를 이용한 반응형 iframe 구현입니다.

<div class="responsive-container">
  <iframe src="./embed.html" frameborder="0"></iframe>
</div>

<style>
.responsive-container {
  position: relative;
  width: 100%;
  height: 0;
  padding-bottom: 50%;
  overflow: hidden;
}
.responsive-container iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
</style>

5. 작은 사이즈 (사이드바용)

사이드바나 작은 공간에 사용할 수 있는 축소 버전입니다.

<iframe src="./embed.html?width=400px&height=350px" width="400px" height="350px" frameborder="0"></iframe>

📋 배포 가이드

  1. GitHub Pages 배포: embed.html 파일을 GitHub 저장소에 업로드하고 Pages를 활성화하세요.
  2. URL 변경: 위의 모든 예시에서 ./embed.html을 실제 GitHub Pages URL로 변경하세요.
  3. 예시: https://username.github.io/repository-name/embed.html
  4. 테스트: 다양한 파라미터 조합을 테스트해보세요.

🔧 URL 파라미터 옵션

  • theme: light (기본값) 또는 dark
  • width: CSS 단위 (예: 100%, 800px)
  • height: CSS 단위 (예: 600px, 50vh)
  • controls: true (기본값) 또는 false
